域名解析

  • windows 下直接在 cmd 里 Ping 一下自己的博客就会得到自己 github 的二级域名的 IP地址：

  • 进入阿里云的管理控制台-域名与网站-云解析 DNS，进入域名的解析设置，点击新手指导，将得到的 IP 地址填到记录值一栏，点击确定就 OK 了。记录值就是自己 github 的二级域名的 IP地址。
